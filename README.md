# CyberQuestsSpring2026
This is my attempt at CyberQuests competition spring 2026
## Overview
This project contains my work for the CyberQuests Spring 2026 challenge. 
The goal of this activity was to analyze network traffic using Wireshark and answer a series of questions related to port scanning, protocol identification, and web authentication.


# I worked through multiple packet capture files and used Wireshark to
- Identify open and closed ports using SYN, SYN-ACK, and RST packets  
- Analyze scan behavior and estimate how long scans took  
- Identify services running on specific ports  
- Analyze industrial protocols like EtherNet/IP (CIP/ENIP)  
- Examine communication patterns between devices  
- Inspect HTTP traffic and extract authentication information
